Формат: csv файл
Поля:

is_canceled: 1 = бронирование отменено, 0 = не отменено (обязательное)

booking_id: уникальный идентификатор бронирования (обязательное)

lead_time: количество дней между бронированием и заездом (обязательное)

arrival_date_year: год планируемого заселения (обязательное)

arrival_date_month: месяц планируемого заселения (обязательное)

arrival_date_week_number: номер недели планируемого заселения

arrival_date_day_of_month: день планируемого заселения (обязательное)

stays_in_weekend_nights: количество забронированных ночей в выходные дни (обязательное)

stays_in_week_nights: количество забронированных ночей в будние дни (обязательное)

adults: количество взрослых проживающих

children: количество детей

babies: количество младенцев

meal: тип питания

country: страна проживания клиента

market_segment: сегмент рынка

distribution_channel: канал, через который было совершено бронирование

previous_cancellations: количество предыдущих отмен

previous_bookings_not_canceled: количество ранее не отмененных броней

reserved_room_type: тип номера, забронированный клиентом

booking_changes: количество изменений в бронировании

agent: турагенство

company: идентификатор компании, совершающей бронирование

days_in_waiting_list: количество дней в листе ожидания до оформления брони

customer_type: тип клиента

adr: средняя стоимость проживания за сутки

required_car_parking_spaces: количество необходимых парковочных мест

total_of_special_requests: количество дополнительных запросов

add_info: дополнительная информация для дальнейшего отображения
